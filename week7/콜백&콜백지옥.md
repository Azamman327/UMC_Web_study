## 동기
- 순서대로 일을 처리하는 방식
## 비동기
- 순서대로 일을 처리하지 않고 빨리 처리 가능한 것부터 먼저

### 동기비동기 처리순서 예시
```js
console.log('1')
setTimeout(function() {
  console.log('2');
}, 10)
console.log('3')
setTimeout(function() {
  console.log('4');
}, 5)

//출력순서 : 1, 3, 4, 2
```
- setTimeout
  + 동기 처리가 다 끝나고 실행됨, 위의 '2'는 동기 처리가 다 끝난 후 10ms후에 실행

## 콜백 쓰는 이유
- 동기, 비동기에서 순서 보장 가능.
